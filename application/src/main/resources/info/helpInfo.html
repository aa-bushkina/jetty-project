Help

For GET all products, you need to go to

http://localhost:3466/products/all

For GET all products by organization, you need enter parameter and to go to

http://localhost:3466/products/byorganization?organization={organization}

For POST request (add new product), you need enter the parameters in the format

http://localhost:3466/products/add?name={name}&organization={organization}&amount={amount}

For POST request (delete product), you need enter the parameters in the format

http://localhost:3466/products/delete?name={name}

where

name - name of product,
organization - id of organization,
amount - amount of product

Initially, there are products:
product1, product2, ... product15
and organizations:
organization1, organization2 ... organization5
in the tables
