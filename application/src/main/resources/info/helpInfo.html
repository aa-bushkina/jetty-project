Help

For GET request, you need to go to

http://localhost:3466/products

For POST request , you need enter the parameters in the format

http://localhost:3466/products/?name={name}&organization={organization}&amount={amount}

where

name - name of product,
organization - id of organization,
amount - amount of product

Initially, there are products:
product1, product2, ... product15
and organizations:
organization1, organization2 ... organization5
in the tables
